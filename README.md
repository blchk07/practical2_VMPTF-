**Практична робота №2**

Робота з JSON, fetch() та React

Метою роботи було ознайомлення з форматом JSON, отримання практичних навичок роботи з мережевими запитами за допомогою методу fetch(), а також створення веб-додатку з використанням бібліотеки React.

На першому рівні було створено HTML-сторінку, яка завантажує JSON-файл зі списком покупок за допомогою fetch(). Отримані дані було розпарсено та відображено на сторінці у вигляді списку елементів. Для цього використовувались стандартні методи роботи з DOM, зокрема створення елементів через createElement.

На другому рівні було реалізовано сторінку для фільтрації JSON-даних. Дані завантажуються з файлу, після чого користувач може ввести значення фільтра. Відповідно до введеного значення відбувається фільтрація масиву об’єктів, і результат відображається на сторінці

На третьому рівні було створено веб-додаток на React для роботи зі списком задач. Дані зберігаються у стані компонента за допомогою useState. Було реалізовано відображення списку задач та можливість фільтрації за їх статусом

На четвертому рівні функціонал додатку було розширено. Додано можливість пошуку задач за назвою, а також реалізовано сортування даних за різними критеріями. Обробка даних здійснюється за допомогою методів масивів, таких як filter, includes та sort, що дозволяє динамічно змінювати відображення без зміни початкових даних.
